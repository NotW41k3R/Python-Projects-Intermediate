# Name the Tracks (F1 Edition)

Completed on Day 25 of 100 Days of Python

A Formula 1 themed guessing game where the player names race tracks from 2024 Calender. Each correct answer displays the track name at its corresponding location on an image with all Track Layouts. The game continues until all tracks are correctly guessed or the player exits.

This version is built using the **Turtle graphics module** and **Pandas** for data handling

**Skills learned:** Data handling with Pandas, file I/O, working with the Turtle library, and integrating data-driven logic into graphical applications.
